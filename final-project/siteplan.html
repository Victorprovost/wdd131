Site Plan

Site Name

Name: Institute Access Enroll

Reason: This name was selected because it clearly represents the website's purpose of facilitating enrollment in institute classes within the LDS community. It is straightforward and easy to remember, making it effective for branding and visibility.

Optional Domain Availability: instituteenroll.org (Check availability)

Site Purpose

The site aims to streamline the registration process for institute classes by collecting information such as email addresses, names, and phone numbers. It will serve as a central hub for potential students to sign up for classes, find information about different courses, and get updates from the institute student council.

Scenarios

  • How can I sign up for an institute class?
  • What courses are available this semester?

Color Schema

Primary Color: Navy Blue

This color will be used for headings and important accents.

Secondary Color: Light Gray

This color will be used for backgrounds and secondary elements.

Typography

Primary Font: Arial

This font will be used for the body text and general content.

Secondary Font: Georgia

This font will be used for headings to create a distinct contrast and improve readability.
